How to replace a offline HDD in v7000
 最近处理了一个v7000换硬盘的case,考虑到今后兄弟们可能会接触到,所以share一下处理的过程和经验. [故障内容] 客户报修,到现场发现有5块硬盘灯不亮,进入到GUI界面去查看发现5块硬盘状态为"脱机"(offline). [开始干活] 参考了v7000的维护guide. 发现通过走fix procedure无法解决问题. 回来了开了PMH(25561).TSG给了一系列的操作步骤,了解到当硬盘状态为脱机的时候,应该通过内部磁盘管理去进行重新激活"被脱机"的硬盘. 引用PMH内容: Take these actions for each reseated offline drive one by one. - Go to the Physical Storage Internal panel in the GUI - Select the drive which was reseated, and select Actions - Mark as - Candidate - Select the drive which was reseated, and select Actions - Mark as - Spare 标记为"未使用"状态以后,硬盘开始重新激活并成为可管理状态.
